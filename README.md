# 스피챗(Speechat)
Automatic translation chat program with KAKAO API

본 프로젝트는 웹 소켓과 카카오 API를 활용한 자동번역 채팅프로그램 제작에 관한 프로젝트입니다. 언어 장벽을 극복한 대화를 추구합니다.

# 사용법

1. ID 입력(한번 정해진 ID는 브라우저 변경이 없는 한 유지)
2. 번역 대상 언어 설정
3. 메시지 입력 후 보내기
4. 텍스트창에서 메시지 확인
